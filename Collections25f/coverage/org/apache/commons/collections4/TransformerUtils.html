38    /**
39    * <code>TransformerUtils</code> provides reference implementations and
40    * utilities for the Transformer functor interface. The supplied transformers are:
41    * <ul>
42    * <li>Invoker - returns the result of a method call on the input object
43    * <li>Clone - returns a clone of the input object
44    * <li>Constant - always returns the same object
45    * <li>Closure - performs a Closure and returns the input object
46    * <li>Predicate - returns the result of the predicate as a Boolean
47    * <li>Factory - returns a new object from a factory
48    * <li>Chained - chains two or more transformers together
49    * <li>If - calls one transformer or another based on a predicate
50    * <li>Switch - calls one transformer based on one or more predicates
51    * <li>SwitchMap - calls one transformer looked up from a Map
52    * <li>Instantiate - the Class input object is instantiated
53    * <li>Map - returns an object from a supplied Map
54    * <li>Null - always returns null
55    * <li>NOP - returns the input object, which should be immutable
56    * <li>Exception - always throws an exception
57    * <li>StringValue - returns a <code>java.lang.String</code> representation of the input object
58    * </ul>
59    * All the supplied transformers are Serializable.
60    *
61    * @since 3.0
