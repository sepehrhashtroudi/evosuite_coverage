31    * Decorates a <code>Map</code> to evict expired entries once their expiration
32    * time has been reached.
33    * <p>
34    * When putting a key-value pair in the map this decorator uses a
35    * {@link ExpirationPolicy} to determine how long the entry should remain alive
36    * as defined by an expiration time value.
37    * </p>
38    * <p>
39    * When accessing the mapped value for a key, its expiration time is checked,
40    * and if it is a negative value or if it is greater than the current time, the
41    * mapped value is returned. Otherwise, the key is removed from the decorated
42    * map, and <code>null</code> is returned.
43    * </p>
44    * <p>
45    * When invoking methods that involve accessing the entire map contents (i.e
46    * {@link #containsKey(Object)}, {@link #entrySet()}, etc.) this decorator

51    * thread-safe.</strong> If you wish to use this map from multiple threads
